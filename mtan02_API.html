Homework Responses

  • Describe the API you selected and what it does: This API provides information on objects from the art collection of the Rijksmuseum in Amsterdam, the Netherlands, including identification details for the object as well as the kind of information that visitors to the museum would like to see such as title, date, artist name, medium, and images of the object.
  • Cite the website where you found it: https://data.rijksmuseum.nl/object-metadata/api/
  • Describe the options you used for the API request: I used the URL "https://www.rijksmuseum.nl/api/nl/collection?key=[apikey]&involvedMaker=Vincent+van+Gogh" for my API request, where [apikey] was replaced with the key I obtained by signing up for an account with the Rijksmuseum website. After the '&'' symbol, the only parameter I specified was "involvedMaker" so that I could choose all works in the collection databse that had van Gogh as the artist. Besides this, I selected a few key-value pairs from the actual JSON object I received through my request, specifically the title and image URL of the array of artObjects that comprised the body of work by van Gogh in the collection.
  • Give two applications where this API would be helpful:
    1. An application that helps users explore artworks in the museum by random generation of a new artwork when a button is pressed or the page is refreshed.
    2. An application that allows users to filter artworks in the museum by metadata such as title and the creator of the work, and the description of the work from the museum.
